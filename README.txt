# Lab5: Работа с таймером на NIOS II

## Описание проекта
Разработана система на процессоре Nios II с использованием аппаратного таймера. Программа циклически выводит на светодиоды значения 0x03, 0x0c, 0x30, 0xc0, сопровождая вывод сообщениями в консоль. Выведена в консоль частота системного таймера и число затраченных тиков. В ходе работы созданы три варианта программы (с использованием alt_nticks(), с таймером callback и с alt_timestamp()).
    
## Аппаратная платформа
- Плата: DE10-Lite (MAX10 10M50DAF484C7G)
- FPGA: Intel MAX 10

## Инструменты
- Quartus Prime 18.1
- Platform Designer
- Nios II SBT for Eclipse
